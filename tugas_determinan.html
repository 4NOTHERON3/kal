Determinan dan Invers Matriks

A. Menghitung Determinan dengan Ekspansi Baris

Rumus ekspansi baris ke-i:

$$\sum_{k=1}^{n} (-1)^{i+k} a_{ik} M_{ik}$$

dengan \(M_{ij}\) adalah minor dari matriks \(A\) dan \(M_{ij} = \det A_{ij}\)

\(A_{ij}\) adalah submatriks dengan menghapus baris \(i\) dan kolom \(j\) dari matriks \(A_{m \times n}\) dengan \(1 \leq i, j \leq n\)

Soal 1 — Determinan Matriks 2×2

Soal 1 — Determinan Matriks 2×2

$$A = \begin{bmatrix} -7 & -5 \\ 1 & 4 \end{bmatrix}$$

Proses Ekspansi Baris 1:

$$\det(A) = (-1)^{1+1} a_{11} M_{11} + (-1)^{1+2} a_{12} M_{12}$$

$$= (+1)(-7)(4) + (-1)(-5)(1)$$

$$= -28 + 5 = -23$$

print(f"det(A1) numpy  = {round(det_numpy)}")

Soal 2 — Determinan Matriks 3×3

Soal 2 — Determinan Matriks 3×3

$$A = \begin{bmatrix} 0 & 2 & -3 \\ 1 & -2 & -1 \\ 0 & 0 & 1 \end{bmatrix}$$

Proses Ekspansi Baris 1:

$$\det(A) = (-1)^{1+1}(0)\,M_{11} + (-1)^{1+2}(2)\,M_{12} + (-1)^{1+3}(-3)\,M_{13}$$

Karena \(a_{11}=0\), suku pertama = 0.

$$M_{12} = \det\begin{bmatrix}1 & -1 \\ 0 & 1\end{bmatrix} = (1)(1) - (-1)(0) = 1$$

$$M_{13} = \det\begin{bmatrix}1 & -2 \\ 0 & 0\end{bmatrix} = (1)(0) - (-2)(0) = 0$$

$$\det(A) = 0 + (-1)(2)(1) + (+1)(-3)(0) = -2$$

A2 = np.array([[ 0,  2, -3],
               [ 1, -2, -1],
               [ 0,  0,  1]])

def minor(M, i, j):
    """Menghitung minor M_ij: hapus baris i dan kolom j (0-indexed)"""
    sub = np.delete(np.delete(M, i, axis=0), j, axis=1)
    return np.linalg.det(sub)

# Ekspansi baris 1 (indeks 0)
det_manual2 = 0
i = 0
for k in range(3):
    cofactor = ((-1)**(i+k)) * A2[i, k] * minor(A2, i, k)
    print(f"  k={k+1}: (-1)^({i+1}+{k+1}) * {A2[i,k]} * M{i+1}{k+1} = {cofactor:.4f}")
    det_manual2 += cofactor

print(f"\ndet(A2) manual = {round(det_manual2)}")
print(f"det(A2) numpy  = {round(np.linalg.det(A2))}")

  k=2: (-1)^(1+2) * 2 * M12 = -2.0000
  k=3: (-1)^(1+3) * -3 * M13 = 0.0000

B. Invers Matriks dengan Rumus Adjoin

Rumus:

$$(\text{adj}\, A)_{ij} = (-1)^{i+j} M_{ji}$$

$$A^{-1} = \frac{1}{\det A} \, \text{adj}\, A$$

Python
def adjoin(M):
    """Menghitung matriks adjoin (transpose dari matriks kofaktor)"""
    n = M.shape[0]
    cofactor_matrix = np.zeros((n, n))
    for i in range(n):
        for j in range(n):
            cofactor_matrix[i, j] = ((-1)**(i+j)) * minor(M, i, j)
    return cofactor_matrix.T  # adjoin = transpose kofaktor

def invers_adjoin(M):
    det = np.linalg.det(M)
    if abs(det) < 1e-10:
        return None, det
    adj = adjoin(M)
    return adj / det, det
Soal 4 — Invers Matriks 2×2

Soal 4 — Invers Matriks 2×2

$$A = \begin{bmatrix} -7 & -5 \\ 1 & 4 \end{bmatrix}$$

Langkah-langkah:

  1. \(\det(A) = (-7)(4) - (-5)(1) = -28 + 5 = -23\)
  2. Kofaktor:
    • \(C_{11} = (+1)(4) = 4\)
    • \(C_{12} = (-1)(1) = -1\)
    • \(C_{21} = (-1)(-5) = 5\)
    • \(C_{22} = (+1)(-7) = -7\)
  3. \(\text{adj}(A) = \begin{bmatrix} 4 & 5 \\ -1 & -7 \end{bmatrix}\)
  4. \(A^{-1} = \dfrac{1}{-23} \begin{bmatrix} 4 & 5 \\ -1 & -7 \end{bmatrix}\)
